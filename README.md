
# Game Show V1

Estrutura inicial do sistema de Game Show ao vivo.

## Tecnologias
- Electron
- React
- Socket.IO
- TailwindCSS

## Funcionalidades V1
- Perguntas em tempo real
- Votação por grupo
- Cronômetro
- Primeiro a votar
- Resposta correta destacada
- Ranking dos grupos
- Tela do operador
- Tela da TV
- Tela mobile

## Estrutura
- server/ -> servidor realtime
- client/ -> interface React
